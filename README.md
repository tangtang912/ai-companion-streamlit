AI智能对话伴侣（Streamlit大模型实战项目）

项目简介

本项目是独立完成的AI大模型应用实战，使用Python开发，基于Streamlit搭建轻量化Web可视化交互界面，对接DeepSeek大模型API，整合LangChain相关开发思路，实现自定义人设对话、流式打字输出、对话会话持久保存功能。
全程自主完成代码编写、环境调试，剔除课程练习冗余代码，仅保留完整可运行成品，用于AI应用开发岗位求职展示。

技术栈

• 编程语言：Python 3.10+

• Web可视化框架：Streamlit

• 大模型服务：DeepSeek-V4-Pro API

• 核心能力：LLM流式对话输出、会话记忆持久化

• 环境管理：requirements.txt统一管理第三方依赖

核心功能

1. 自定义AI伴侣人设
可自定义AI昵称、性格、说话风格，通过System Prompt约束大模型输出语气，打造专属对话角色。

2. 流式实时对话输出
采用chunk分片解析接口返回内容，模拟真人逐字打字聊天效果，交互体验流畅。

3. 会话记忆保存
完整记录每一轮用户与AI对话，页面刷新后历史聊天记录不会丢失，支持长上下文连续对话。

4. 轻量化一键启动
无需复杂前端部署，仅安装依赖后一行命令即可拉起网页客户端，上手门槛低。
